Shiny apps hosting

Shiny allows users of R to create web applications without learning HTML, CSS or JavaScript - learn more at the dedicated Shiny website.

The IDN will host Shiny apps developed by (or on behalf of) Oxford researchers provided they meet the following requirements:

  • The Shiny app visualises/explains research outputs of Oxford researchers
  • All data displayed by the app is deposited in (and accessed from) a DOI issuing data repository
  • The code for the Shiny app is also deposited in a DOI issuing data repository
  • The Shiny app contains the DOI for the underlying code and data

Please visit the dedicated IDN Showcase for examples of Shiny apps built and hosted by the IDN.
